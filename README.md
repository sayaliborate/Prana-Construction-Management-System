 About the Project

Prana Construction Management System is a full-stack web application built to simplify and streamline construction project management.
It enables teams to manage projects, tasks, and users efficiently with a clean and responsive interface.

This project focuses on real-world construction workflows and role-based access control.


 Features

 User Authentication & Authorization
 Role-Based Access (Admin / User)
 Project & Task Management
 Progress Tracking
 Clean & Responsive UI
 Secure Data Handling
